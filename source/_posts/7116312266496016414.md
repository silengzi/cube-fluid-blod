vue中默认@表示src文件夹的路径

①导入js文件时可以：

```js
import '@/store/index.js'
```

②导入css文件时@符号前面要加一个~符号才可以:

```css
import '~@assets/css/test.css'
```

③html中使用时直接用@就可以:

```html
<img src="@/assets/img/test.png" alt="error">
```
